# 系统简介

## 用户登录

- 网址：调度运行指挥平台网址为：http://10.41.4.60:8083/netorder  。

- 浏览器：建议采用Google Chrome（谷歌浏览器）或国网安全浏览器登入。

- 账号：默认使用身份证号（该身份证号需在OMS持证上岗考试中获取过证书）。220kV变电站运维人员使用本运维班账号登录。

    >提醒：
    >凡是通过省调持证上岗考试的人员，即在省调OMS中可以查询到证书的人员，均具有网络化下令系统账户。无需单独申请。

- 密码：初始默认统一登录密码。可联系省调专责询问。

## 用户退出

点击退出当前用户，可点击右上角“电源”按钮。

## 个人中心

点击**右上方本人名字**可查看个人中心，个人中心包括刷新、注销、网络正常、常用下载、接班、修改密码、静音、发布说明等功能。

- 刷新：若发现操作票签收无按钮，可尝试清理浏览器缓存（Ctrl+shift+del）后刷新。

- 注销：当本人不再操作此系统需及时注销。

- 网络正常：可查询各单位当前登录用户。

- 常用下载：目前提供了Chrome浏览器下载连接和**国网系统内**使用本系统所需防火墙开通流程。

    >提醒：
    >在此处可以找到申请防火墙开通流程的说明文档，下载按照流程申请即可。

- 接班：交接班后需及时登记本班值班员。

- 修改密码：默认密码可按需修改。

## 常见问题

### 1. 网页不刷新，找不到要签收的票或者预令

可能原因：浏览器缓存问题。

解决方法：清理浏览器缓存（Ctrl+shift+del）后刷新，重新登陆。

### 2. 找不到页面右上角的`提交`或`保存`按钮

可能原因：系统分辨率过高或过低。

解决方法：调整系统分辨率，按CTRL+鼠标滚轮缩放，将系统分辨率调整至50%-200%进行观察。

### 3. 220kV变电站电脑无法进入网络化下令网页

可能原因：未申请开通防火墙。

解决方法：按照防火墙开通申请流程（`常用下载`中可下载），申请开通防火墙。

### 4. 220kV运维班管辖的变电站变了，签收的运维班需要调整怎么办

解决方法：一个运维班的账号只能签收预发至本班运维的站的逐项票。先确认该变电站对应的新运维班的电脑能否登录系统。  
若能登录，但收不到省调的预发逐项票，联系省调专责修改系统后台逻辑。  
若不能登录，先按前文流程，申请开通防火墙。

### 5. 本单位新进了几名人员，如何为他们开通系统账号？

解决方法：首先确认新进人员已在省调OMS中完成持证上岗考试。在系统中可以查其是否有持证上岗证书。有证书则必有账号。    
若已有证，则无需申请，本系统定期同步人员信息。若无证，则先参加持证上岗考试。
