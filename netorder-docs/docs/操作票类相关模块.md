# 操作票类相关模块

## 逐项操作

### 签收预令

收到省调通知，需签收预令时，则：

1. 点击系统左侧菜单，依次点击`逐项操作`-`厂站预发中`。

2. 在右侧页面中选择省调通知签收的预令票。

3. 进入需签收的票面后，认真核对本场站操作项。

4. 若无疑问，则在本操作票页面中，点击右上方`签收`按钮进行签收。

5. 根据公司“双下达”要求，220kV站运维人员与地市集控人员均需签收预令。

### 操作执行

所有正在进行的操作票，均可在`逐项操作`-`厂站执行中`找到，找到后进入对应的票面：

找到票面中正在操作的指令项目，进入处理。网络发令环节有：
   
   * 调度发令
   
   * 厂站复诵
   
   * 调度确认
   
   * 厂站回令
   
   * 调度收令
   
   * 厂站确认

在执行调度指令时需电话-网络双轨制开展，所有复诵、回令操作均需要使用调度电话确认。

### 归档查询

用户点击`逐项操作`-`厂站归档`，则可以查看执行完后归档的操作票。

## 许可操作

### 操作申请

1. 点击系统左侧菜单，依次点击`许可操作`-`拟票`。

2. 页面将显示“`新增一次操作`”、“`新增二次操作`” 、“`新增机组操作`”三种操作票类型，用户根据需求拟相应的操作票。下流程以“新增一次操作”举例。

3. 进入一次操作命令票后，首先需填写基本信息。申请操作类型为默认停送电，停电和送电会在一张票上完成，常规情况下提停送电票；特殊情况下也可以单独提交停电或者送电操作票类型。
   
   > 注意： 如有OMS日计划，则尽可能进行关联，减少填写票面基本信息工作量。如为临时操作，应手动将票面基本信息填写完整。

4. 基本信息填写完成后，在下方`停电信息`栏目中，点击`新增停电操作`按钮，选择正确的主设备，在操作内容栏目下方将所需操作指令填写清楚，同时注意备注栏的内容填写。

5. 所有信息填写完成后，点击右上角`停电送审`并电话申请操作。

### 操作审核

审核环节是由调度员处理的环节，厂站端人员无需操作此环节。本票右上角可以查看状态，如果是停电送审当前状态是等待停电审核，如果是送电送审当前状态是等待送电审核。

### 操作执行

厂站人员首先找到本票面对应的`停电信息`/`送电信息`栏，找到`指令操作`项目，在下方点击`处理`按钮进行操作。

正常操作步骤分为5步：

* 调度复诵

* 厂站确认

* 调度下令

* 厂站回令

* 调度收令

正常按流程操作即可。

### 停运设备

对于**停送电**操作票，在所有停电指令完成后状态变成停电已完成。该命令票自动流转到停运设备菜单中。

在该页面中进行送电申请操作，点击“`新增送电操作`”填写相应的信息。注意至少有一条设备是主设备。

将相应信息填写完整后点击右上角“`保存`”保存表单信息，点击右上角“`送电送审`”保存并发送至审核环节。

> 注意：在申请某设备送电时，优先在`停运设备`中寻找停送电流程执行后的表单，减少基本信息填写工作量，并实现流程闭环。

### 归档

在送电完成后由省调调度员进行归档，厂站端人员可以查看归档票。

若某停送电流程执行完停电流程后送电流程无需执行（如线路π接后遗留的原设备停电流程），则需电话汇报省调值班调度员，手动进行归档。

## 地线库登记

许可操作的220kV线路完成后，系统将自动弹窗提示进行地线登记。必要时也可手动新增。

* 设备范围：只登记省调调度且为省调调规中操作权调整的线路，省调许可操作指令完成后登记。

* 设备状态：只登记转检修的线路。如线路仅停运或解备，未作安措，则无需登记。

停电操作完成后需及时进行接地线登记。送电完成后需及时进行拆地线登记。

## 单项操作

### 预发

对于调规中规定应使用单项令的操作，使用单项操作模块开展。最常用的情景是保护及稳控装置或通道的投退。

省调需进行单项操作时，将在系统中拟票完毕后，可提前预发至接令人。对于临时或紧急情况，也可不预发直接下令。

接到省调签收预令的电话通知后，需要在`预发`栏目页面中进行查看，若无疑问，则在此进行签收。

### 执行中

对于正在执行的的调度指令票，在此栏目页面中进行执行。

对于票面中单条调度指令执行流程，系统的操作方法与逐项操作类似，共六步，均需与省调电话沟通并双轨制开展。

### 已归档

对于已执行完毕的单项操作指令票，可以在此栏目中查询相关历史信息。
