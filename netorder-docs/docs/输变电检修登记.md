# （新）输变电检修登记

## 简介

应调度处领导要求，为推动OMS日计划中计划闭环，在网络化下令中开发了本模块，可实现已达到调度执行环节的输变电检修计划的相关流程管理，并最终实现日计划的统一闭环管理。

> **注意：  
本登记流程不是检修票  
本登记流程不是检修票  
本登记流程不是检修票  
仅用于OMS日计划中计划流程闭环。流转原则为“谁报送计划，谁负责闭环”。**

## 交互流程

对于某条已批准的日计划流程，正常交互流程为：

1. 下发：省调调度员接到计划处送通知的次日检修计划后，需要将本条计划通过网络化系统下发，默认下发至上报单位，无需像网检票一样现场签收后调度员确认再发送至`待开工`流程。
    >提醒：
    > * 默认下发单位为计划上报单位，若需调整可手动调整。
    > * 批量下发功能正在开发中，计划3月上线。

2. 开工登记：厂站、地调等计划上报单位，在现场开工后，需要及时在系统登记开工。该流程无需调度员手动确认，但可手动退回；

3. 完工登记：计划上报单位，在完成检修后，需要及时在系统登记完工。同样地，该流程无需调度员手动确认，但可手动退回；

4. 归档：计划申报单位完成本单位计划后，调度员确认，手动在网络化系统将本条日计划归档，系统会自动在OMS发送流程，更新为“已归档”。

## 延期操作
当计划申报单位需申请日计划延期时，在系统中填写相关信息并申请，电话汇报省调。

省调调度员接到电话汇报后，可直接点击右上角发送操作，无需再次在OMS中手动填写。

当延期申请开始进入会签流程后，网络化系统将自动取OMS系统中相关信息，无需调度员手动填写，将`延期申请`流程中的日计划条目发送至`现场完工汇报`流程中。

## 常见场景举例

> 提醒：
> 前面几种场景仅涉及一个单位（省调调度员视角），操作可使用许可操作完成，流程相对较为单一。

### 场景1：委托地调调度的110kV及以下新能源场站并网线路检修计划

如：平顶山供电公司报送日计划：110kV古田线停运检修，则流程为：

1. 省调调度员接到次日检修计划后，将本条计划通过网络化系统下发至平顶山地调。地调可在系统查看已下发的检修计划，无需签收。

2. 地调按计划向省调申请`许可操作`，进行转检修操作。

3. 在现场检修工作开工后，地调在在本系统登记开工。省调调度员可以在系统查看已登记的开工计划，无需手动确认。发现有误时可以手动退回。

4. 在现场检修工作完成后，申请送电前，地调在本系统登记完工，同时提出送电的`许可操作`。同样地，省调调度员可以在系统查看已登记的完工计划，无需手动确认。发现有误时可以手动退回。

5. 省调调度员接到汇报后，可将本条日计划发送归档。同时许可地调送电。

### 场景2：省调直调、且为操作权调整设备的220kV一二次设备检修计划；或电厂站内省调直调220kV设备的检修计划

例如：
* 濮阳供电公司报送日计划：沃220kV西母PT检修
* 洛阳供电公司报送日计划：220kV九兴线停运检修
* 焦作供电公司报送日计划：220kV阳洛变退出第二套母差
* 三门峡供电公司报送日计划：220kV虢都变双套稳控装置轮退
* 棉山电厂报送日计划：棉220kV北母西段停运检修

此类场景流程与前文类似，为：

1. 省调下发；

2. 地调/电厂申请`许可操作`，操作完成后现场开展计划检修，向省调登记开工；

3. 检修完工后，地调/电厂在系统中登记完工；同时向省调提`许可操作`送电申请；

4. 省调调度员接到汇报后，可将本条日计划发送归档。同时许可检修后的设备送电。

### 场景3：新设备启动试运行计划，或220kV母线配合停电计划

例如：
* 焦作供电公司报送日计划：220kV阳洛变#2主变启动试运行
* 濮阳供电公司报送日计划：策220kV南母腾空备用，配合新线路启动试运行

流程为：

1. 省调下发至计划上报单位（若为地市供电公司上报，则下发至地调）；

2. 无论新设备启动试运行操作为地调操作或省调操作（即无论省调`许可操作`或省调`逐项操作`），在开始送电操作后，计划下发单位需登记开工；

3. 送电操作完成、开始试运行时，计划下发单位需登记完工。省调调度员接到汇报后，可将本条日计划发送归档。

### 场景4：500kV变电站站内设备检修计划，或送变电公司报送500千伏线路带电作业

例如，超高压公司报送如下日计划：
* 祥5033开关转检修
* 瀛#2主变本体重瓦斯保护退出
* 湛220kV北段母线第二套母线保护退出
* 66kV瀛#8电容器检修

或：送变电公司报送：500kV南香Ⅰ线带电作业

流程为：

1. 省调下发至计划对应500kV站；（带电作业则下发至送变电公司）

2. 网调/省调按计划进行`许可操作`；（带电作业则按需进行重合闸操作）

3. 操作完成后，**如有**`网检票`则按正常流程履行开工手续；然后在本系统中登记开工；（带电作业则需注意同时履行`带电作业票`开工手续）

4. 检修工作完成后，**如有**`网检票`则按正常流程履行完工手续；然后在本系统中登记完工；（带电作业则需注意同时履行`带电作业票`完工手续）

5. 省调调度员接到检修工作完成的汇报后，可将本条日计划发送归档。而后正常开展送电操作（或汇报网调进行操作）。

---

>提醒：
> 下面的例子涉及多个单位，且涉及`省调高压线路检修票`、`网检票`等业务流程，注意防止混淆或遗漏。

## 场景5：省调直接逐项操作的220kV及以上的线路

例如：
* **220kV**：漯河公司报送日计划：220kV湛董线停运检修（漯河公司、平顶山公司有线路工作，董庄变、湛河变均有站内工作）

流程为：
1. 省调下发至计划上报单位（即漯河地调，**不下发至平顶山地调及500kV湛河变**）；

2. 省调按计划进行`逐项操作`；

3. 操作完成后，省调通知平顶山地调、漯河地调操作完成，按正常流程履行`省调高压线路检修票`开工手续；

4. 计划下发单位（即漯河地调）在本系统中登记开工；

4. 漯河公司线路工作完工后，向省调履行`省调高压线路检修票`完工手续。在与董庄变确认站内工作完工后，漯河地调在本系统中登记完工；

5. 待到计划完工时间，省调调度员核实所有线路工作、及所有站内工作已完工，所有`省调高压线路检修票`已终结后，发送本条日计划流程终结。

6. 省调调度员按计划进行`逐项操作`，安排送电。

再例如：
* **500kV**：超高压公司报送日计划：500kV周嵖线停运检修（送变电公司有线路工作，周湾电厂、嵖岈变均有站内工作）

流程为：
1. 省调下发至计划上报单位（即500kV嵖岈变，**不下发至送变电公司及周湾电厂**）；

2. 网调许可后，省调按计划进行`逐项操作`；

3. 操作完成后，省调通知嵖岈变、周湾电厂、送变电公司操作完成，按正常流程履行`网检票`开工手续，与送变电公司按正常流程履行`省调高压线路检修票`开工手续；

4. 计划下发单位（即嵖岈变）在本系统中登记开工；

4. 嵖岈变站内工作完工后，可向省调报`网检票`完工，同时嵖岈变在本系统中登记完工；

5. 待到计划完工时间，省调调度员核实所有线路工作、及所有站内工作已完工，`网检票`已终结、所有`省调高压线路检修票`已终结后，发送本条日计划流程终结。

6. 经网调许可后，省调调度员按计划进行`逐项操作`，安排送电。

## 场景6：上级调度直调的500kV及以上设备

例如直流公司报送日计划：
* 豫南站中州安控装置1、2退出（豫南站站内工作）
* 灵宝站单元Ⅰ检修（灵宝站站内工作）
* 南香Ⅰ线停运检修（南阳站、豫南站站内工作，送变电公司有线路工作）

流程为：
1. 省调下发至计划上报单位（如豫南站或南阳站）；

2. 上级调度按计划进行操作；

3. 相关特高压站及送变电公司按正常流程履行`网检票`开工手续，与送变电公司按正常流程履行`省调高压线路检修票`开工手续（如有）；

4. 计划下发单位（如豫南站或南阳站）在本系统中登记开工；

4. 站内工作完工后，可向省调报`网检票`完工，同时在本系统中登记完工；

5. 待到计划完工时间，所有`网检票`终结后，调度员发送本条日计划流程终结。


6. 上级调度安排送电。
